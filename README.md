## Run Locally

Clone the project

```bash
  git clone <repo Url>
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```



## Environment Variables

To run this project, you will need to add the following environment variables to your .env file Key from MovieDB

`REACT_APP_API_KEY`
